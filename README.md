## 📚 CS Study 목적

이 스터디는 컴퓨터공학(CS)의 핵심 개념을  
**단순 암기가 아니라 이해 중심으로 정리하고**,  
개발자로서 기본 체력을 쌓기 위해 진행한다.

- OS, Network, Database 등 CS 전반을 꾸준히 학습한다
- 공부한 내용을 **내 언어로 정리하는 습관**을 기른다
- 면접 대비뿐만 아니라, 실제 개발에서의 이해를 목표로 한다 🚀

---

## 📝 Study Rules

### ✅ 기본 규칙
- **일주일에 1회**, 계획한 범위까지 공부해오기
- 매주 학습한 내용 중 아래 항목을 반드시 정리해오기 👇  
  - 이해하기 어려웠던 개념 🤔  
  - 중요하다고 생각한 핵심 내용 ⭐  
  - 추가로 찾아본 내용이나 예시 🔍

### ✅ 정리 방식
- 각 주제는 **Markdown(.md)** 파일로 정리한다
- 단순 복붙 ❌  
  → **자신의 말로 설명하기**를 원칙으로 한다 ✍️
